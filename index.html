My Projects

Mini Clone

Mini Clone is a URL shortener application where users can convert lengthy URLs into short links and track the number of clicks. The backend is powered by Spring Boot with JWT-based authentication and Spring Security. The frontend uses React and Vue, styled with Tailwind CSS, and MySQL is used for persistent data storage.

Academic Tool using React JS

A full-stack academic task tracking tool built with React, Node.js, MongoDB, Docker, CircleCI, and Postman. It helps students and researchers manage scholarly projects, track references, and generate academic portfolios or CVs. Features include tagging, activity tracking, API testing, and CI/CD integration. Designed for extensibility, it supports future integrations with platforms like GitHub, Canvas, and Jira.

Smart Content Using Gemini API and Springboot

Smart Content is an AI-powered summarization tool that extracts key insights from paragraphs and generates concise three-line summaries. Built with Spring Boot, HTML, CSS, and JavaScript, it uses the Gemini API for intelligent text processing, enhancing readability and efficiency for professionals, researchers, and students.

Snake Game Using Java

The Snake Game challenges players to guide a snake through levels with increasing speed and obstacles. Eating food boosts the score and speeds up the snake. With three lives, the game ends after all are lost. Each milestone raises difficulty, requiring quick reflexes and strategy to survive and score high.

Nutritional- count using React Native

Nutritional-Count is a React Native mobile app that tracks daily nutrition. I developed and maintained its user-friendly UI and robust backend using Node.js, RESTful APIs, and a database. I used Ngrok for secure API testing, implemented client feedback, and fixed bugs to enhance performance and ensure app stability.
